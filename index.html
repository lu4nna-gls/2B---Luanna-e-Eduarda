Projeto Agrinho

O Projeto Agrinho é um jogo educativo criado com foco no tema "Agricultura Familiar", desenvolvido durante as aulas de Pensamento Computacional e Geografia. O objetivo principal foi promover o aprendizado de forma interativa, utilizando a tecnologia como ferramenta de apoio pedagógico.

O jogo foi desenvolvido em uma plataforma de programação visual, permitindo que os alunos aplicassem conceitos como lógica de decisão, variáveis e interatividade. As perguntas abordam temas relacionados à importância da agricultura familiar para o meio ambiente, economia e segurança alimentar.

O design do jogo foi pensado para ser acessível e atrativo, com elementos visuais simples, botões interativos e pontuação dinâmica. Essa abordagem lúdica torna o aprendizado mais leve e envolvente para os usuários.

Jogo do Pong

O jogo do Pong foi recriado utilizando a plataforma Scratch como parte do conteúdo de Pensamento Computacional. O projeto teve como finalidade explorar conceitos de programação como loops, eventos, colisões e variáveis de forma prática e divertida.

Inspirado no clássico jogo dos anos 70, esta versão foi personalizada com pontuação automática, movimentação dos jogadores e resposta visual ao impacto da bola. O jogo também permite partidas entre dois jogadores, promovendo interação e desafio.

Durante o desenvolvimento, os alunos puderam exercitar a criatividade, além de compreender como funcionam as estruturas lógicas por trás dos jogos digitais. A interface simples e os controles intuitivos garantem uma experiência acessível para iniciantes.

Site para aprender a desenhar

Este site foi o primeiro projeto desenvolvido com HTML, tendo como objetivo oferecer um espaço simples e funcional para quem deseja aprender a desenhar. O conteúdo foi criado com foco em iniciantes, apresentando vídeos tutoriais, dicas básicas e recursos visuais para estimular a prática do desenho.

A estrutura do site foi montada com HTML sem uso de JavaScript ou CSS avançado, destacando o conteúdo de forma limpa e objetiva. Os elementos são organizados em seções temáticas, permitindo fácil navegação e compreensão do usuário.

O desafio deste projeto foi trabalhar com HTML puro e entender a lógica de construção de uma página web. A experiência proporcionou uma base sólida para futuros projetos mais complexos, além de reforçar a importância da clareza e simplicidade no design de interfaces.
